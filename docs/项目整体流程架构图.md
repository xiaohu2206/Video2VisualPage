## 调度与重跑控制

```mermaid
flowchart TD
    Cmd["用户命令<br/>run / resume / stage / rerun / render / qa"]
    Normalize["阶段别名归一化<br/>编号 ID、业务名、命令别名统一到 stage_id"]
    Scope["确定执行范围<br/>stage_slice / parse_stage_range / first_incomplete_stage"]
    Deps["依赖闭包<br/>自动补齐缺失上游步骤"]
    Current{"阶段是否最新<br/>run_state=done<br/>step_manifest=done<br/>依赖产物未更新<br/>模型签名一致"}
    Skip["跳过执行<br/>记录 stage_skipped"]
    Running["标记 running<br/>写 run_state.json"]
    Runner["执行阶段 Runner<br/>video2visualpage/stages/*.py"]
    Success["阶段成功<br/>收集 outputs"]
    Failed["阶段失败<br/>记录错误信息"]
    Manifest["写 step_manifest.json<br/>stage_id / step_name / status / outputs / json_outputs / result"]
    Done["标记 done<br/>更新 run_state.json"]
    ErrorState["标记 failed<br/>写 step_manifest failed<br/>写 logs/errors.jsonl"]

    Cmd --> Normalize --> Scope --> Deps --> Current
    Current -- 是 --> Skip
    Current -- 否或强制重跑 --> Running --> Runner
    Runner -- 成功 --> Success --> Manifest --> Done
    Runner -- 异常 --> Failed --> ErrorState
```
